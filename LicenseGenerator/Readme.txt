[Title]

Innolas License Generator

[Version] 

Version 1 Beta


[Headline]

This program is made to create a license .dat files for the PLC machines license.


[Description]

To use this program you have create first the machine under Manage Machines. Hier you will need 
to write down the Pxxx number, Customer name and Mac Address of the Network card which 
is connected to the PLC clamps. In case the machine is using a CX unit, please use Mac address 
of this.
